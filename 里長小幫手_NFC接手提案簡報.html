舊社里里長小幫手 / 官方 LINE 接手提案 01 / 09
LINE 接手 + NFC 宣傳入口

讓里民找得到、用得懂、留得住

先把官方 LINE 的功能與資料流盤點清楚，再把服務整理成一個入口；NFC 卡片與鑰匙圈負責把入口帶到每一位里民手上。

為什麼要先接手

現在的重點不是重做，是先把控制權拿回來

官方 LINE 已經是里民最容易接觸到的服務入口。接手時要確認每個按鈕導去哪裡、資料送給誰、後續誰能處理。

確認現有圖文選單逐一盤點按鈕、網址、關鍵字、LIFF、表單與回覆訊息。
確認資料歸屬姓名、電話、陳情內容、照片上傳等資料，不應留在不明帳號或前任廠商手上。
確認可持續維護活動、商店、公告、報名名單，要能由里辦或授權人員自己更新。
第一階段

官方 LINE 功能盤點表

先把目前的功能畫成一張清楚的表，里長就能知道哪些能保留、哪些要改、哪些有資料風險。

01 圖文選單截圖

確認每個按鈕區塊、名稱、位置與目前呈現方式。

02 實際點擊測試

用使用者角度看回覆、跳轉網頁、表單是否可用。

03 後台與權限確認

檢查網址、LINE 後台、Webhook、表單回覆資料歸屬。

04 接手清單

列出保留、修正、重建、停用的功能與優先順序。

4

先接四個最常用服務

不要一開始塞滿功能。先讓里民最常做的事情變簡單，使用率會比功能數量重要。

活動報名與查詢報名、確認、候補、現場報到，減少人工對名單。
里民通報 / 陳情拍照、描述、聯絡方式、處理狀態，讓案件可追蹤。
公告與社群入口官方 LINE、LINE 社群、Facebook、Instagram 集中在同一頁。
特約商店 / 美食地圖把地方店家與里民服務連起來，做出可感受的在地價值。
第二階段

NFC 宣傳卡與鑰匙圈：把服務入口帶到現場

NFC 不直接存個資，只寫入一個可控的入口網址。里民手機碰一下，就能加入官方 LINE、社群、FB、IG 或打開報名與通報。

服務處入口牌

櫃台放一張「手機靠這裡加入官方 LINE」，長輩不用搜尋帳號。

活動現場報到

志工拿 NFC 牌或鑰匙圈，里民碰一下打開報到/報名頁。

公告欄數位入口

公告欄貼 NFC + QR Code，碰一下看最新公告、活動與停水停電資訊。

特約商店導流

合作店家放 NFC 貼紙，導到優惠、地圖、店家介紹與官方 LINE。

里民通報點

公園、巷口、常見問題點貼 NFC，碰一下直接開拍照通報。

里長電子名片

鑰匙圈、名片、活動小卡，都導到同一個舊社里服務入口。

成本抓法

先小量測試，不急著大量印製

第一批目標是驗證手機感應、頁面內容、加入 LINE 流程與現場使用方式。確認後再做正式彩印版。

測試版 NT$1,000-3,000 20-50 個空白 NFC 鑰匙圈或貼紙，用手機寫入，搭配簡易貼紙標示。
里辦試辦版 NT$3,000-10,000 50 個鑰匙圈、200 張 NFC/QR 小卡、20 張貼紙，足夠服務處與活動測試。
正式推廣版 依數量報價 彩印卡片、客製鑰匙圈、抗金屬貼紙，可交由廠商印製，里辦掌握入口網址。
執行原則

實體物可以固定，內容必須能更新

NFC 卡片寫入後最好鎖定，避免被覆寫；但鎖定前要寫的是「舊社里入口頁」，不是單一 LINE 或 FB 連結。

卡片只寫入口網址例如 jiusheli.tw/card。之後社群、官方 LINE、活動連結都在網站後台調整。
NFC + QR Code 一起印有些手機 NFC 沒開或不會感應，QR Code 是必要備援。
測試版先不要鎖確認流程沒問題後，正式版再設定唯讀，避免被惡意改寫。
里長可以怎麼說
「我們不是只做一張電子名片，而是把里民服務入口整理好，讓大家碰一下就找到里長。」

這句話很重要：NFC 是宣傳工具，真正的價值是官方 LINE 接手、資料可控、服務流程可追蹤。

建議下一步

先做盤點，再做入口頁，最後做 NFC 試辦

一週內可以先完成官方 LINE 現況盤點與試辦入口頁；第二週開始小量 NFC 實測，活動或服務處就能看見效果。

提案執行順序

  • 盤點官方 LINE 圖文選單與現有網址
  • 確認表單資料與後台權限
  • 整理舊社里服務入口頁
  • 製作 NFC/QR 試辦卡與鑰匙圈
  • 服務處與活動現場實測後再正式印製
